# Happy Birthday Surprise

A birthday surprise website built using HTML, CSS, and JavaScript.

## Pages

- `index.html` — landing page with animated background, stars, balloons, hearts, sparkles, and a start button.
- `cake.html` — birthday cake animation with candles, fireworks, confetti, and a continue button.
- `video.html` — video memory page with a blurred background video and skip button.
- `letter.html` — animated letter reveal with a typed birthday message and replay button.

## How to use

1. Open `index.html` in Live Server or any web server.
2. Click **Open Surprise** to move to `cake.html`.
3. On `cake.html`, click the candles to start the celebration.
4. Continue to the video page, then wait or click **Skip**.
5. The final letter page contains the birthday note and replay option.

## File structure

- `index.html`
- `cake.html`
- `video.html`
- `letter.html`
- `css/` — styles for each page and shared layout.
- `js/` — page controllers and effects.
- `assets/images/` — memory photos.
- `assets/music/` — background and birthday music.
- `assets/videos/` — birthday video.

## Notes

- All interactions are handled with vanilla JavaScript.
- The project requires a web server for correct relative asset loading in some browsers.
- If a page does not work, verify the referenced asset paths.
